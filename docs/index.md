## Acerca de este proyecto

Este proyecto demuestra la integración de múltiples herramientas y tecnologías en un flujo de trabajo coherente:

- **Git y GitHub**: Control de versiones y colaboración

- **MkDocs**: Generación de sitios web estáticos para documentación
- **GitHub Actions**: Automatización CI/CD
- **GitHub Pages**: Hosting estático gratuito
- **Docker y NGINX**: Contenedorización y servidores web
